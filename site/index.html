sender-trades

0DTE intraday options trading recommendation system for SPY and QQQ. Ingest, analyse, decide, execute — on your own infrastructure.

● 63 tests ● 3 strategies ● 8 risk guardrails
Get started View on GitHub

Features

📈

Momentum strategy

Detects gap openings and intraday momentum breaks on SPY and QQQ.

↔️

Mean-reversion strategy

RSI-based contrarian signals when the market over-extends.

⚡

Event-driven strategy

Catalyst-aware recommendations from news sentiment and macro data.

🧩

Decision aggregator

Picks the highest-confidence recommendation or merges near-ties with 2-of-3 consensus.

🛡️

8 risk guardrails

Max loss cap, bid/ask spread check, time windows, OCC chain validation, statistical sanity, and more.

🔌

MCP broker abstraction

Subprocess stdio transport for Alpaca and Robinhood. Dry-run by default — nothing leaves your machine.

📰

Multi-source ingestion

Finnhub quotes, Brave news, Reddit WSB/options sentiment, Unusual Whales options flow, RSS feeds, and Atlas morning briefings.

📋

Structured logging

JSON logs with per-run correlation IDs. Every decision is traceable.

Quick start

git clone <repo> && cd sender-trades
Clone the repository
uv sync --all-extras
Install dependencies
cp .env.example .env
Configure API keys (Finnhub, Brave, Alpaca)
python -m src.main --dry-run
Run a dry run — no trades executed
docker compose run --rm app python -m src.main --dry-run
Or run via Docker
make check
Run lint + format check + tests
docker compose up docs
View Sphinx API docs at localhost:8080
